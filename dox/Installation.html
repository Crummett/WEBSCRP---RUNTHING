Tested with:

Chrome - All works fine. Largely made in Chrome.

Firefox - Loads fine (some formatting issues on forms). Posting does not work at all. No idea why. No error messages generated.

Edge - All works fine, as with Chrome.

Installation

  1. cd into the website’s folder.
  2. Run the command: $ node server.js
  3. If no databases already exist, a new one will be made entitled ‘runthing’. Only one piece of dummy data: a club called ‘Unattached’ containing any users created without a club.
